# LaTeX Templates Quick Reference

Source: SECNAV M-5216.5 (DON Correspondence Manual) | MCO 5216.20B

---

## Universal Requirements (All Documents)

| Element | Requirement |
|---------|-------------|
| **Font** | Times New Roman 12pt preferred (10-12pt acceptable) |
| **Alternative Font** | Courier New (informal only) |
| **Paper** | 8.5" x 11" white bond |
| **Margins** | 1" all sides |
| **Justification** | Left only (NO right/center/full justify) |
| **Spacing** | Single within paragraphs, double between |

---

## Paragraph Structure (8 Levels Max)

Per SECNAV Ch 7, Para 13.b, Figure 7-8:

| Level | Label | Indent |
|-------|-------|--------|
| 1 | `1.` | 0" (left margin) |
| 2 | `a.` | 0.25" |
| 3 | `(1)` | 0.5" |
| 4 | `(a)` | 0.75" |
| 5 | `1.` (underlined) | 1.0" |
| 6 | `a.` (underlined) | 1.25" |
| 7 | `(1)` (underlined) | 1.5" |
| 8 | `(a)` (underlined) | 1.75" |

**Rule:** If "a" exists, must have "b" (no single sub-items)

## Document Types

### Standard Naval Letter
- **Chapter:** Ch 7
- **Letterhead:** YES
- **Numbered Paragraphs:** YES (8 levels)
- **From/To Lines:** YES
- **Signature:** Abbreviated name only (J. M. SMITH), NO rank

### Letterhead Memorandum
- **Chapter:** Ch 7
- **Letterhead:** YES
- **Numbered Paragraphs:** YES (8 levels)
- **Same as Standard Naval Letter**

### Memorandum for the Record (MFR)
- **Chapter:** Ch 7
- **Letterhead:** OPTIONAL
- **Numbered Paragraphs:** YES (8 levels)
- **SSIC:** Optional
- **Use:** Internal record-keeping

### Business Letter
- **Chapter:** Ch 11
- **Letterhead:** YES
- **Numbered Paragraphs:** NO (indent 0.5" instead)
- **Date Format:** Spelled out (January 5, 2015)
- **Salutation:** YES (Dear Mr./Ms.:)
- **Complimentary Close:** YES (Sincerely,)
- **References/Enclosures:** Mentioned in body only, NO formal lines

### Joint Letter / Joint Memorandum
- **Chapter:** Ch 10
- **Letterhead:** Dual (both commands)
- **Numbered Paragraphs:** YES (8 levels)
- **Signature Order:** Junior LEFT (signs first), Senior RIGHT (signs last)

### MOA / MOU
- **Chapter:** Ch 10, Para 6
- **Letterhead:** Both commands or plain bond
- **Numbered Paragraphs:** YES (8 levels)
- **Signature:** Overscored names (line above)
- **Signature Order:** Junior LEFT (signs first), Senior RIGHT (signs last)

### Endorsement (Same-Page)
- **Chapter:** Ch 9
- **Letterhead:** N/A (continues on basic letter)
- **Numbered Paragraphs:** NO (continues basic letter sequence)
- **Use:** When endorsement fits on basic letter's page

### Endorsement (New-Page)
- **Chapter:** Ch 9
- **Letterhead:** YES
- **Numbered Paragraphs:** NO (continues basic letter sequence)
- **Must Repeat:** Subject line from basic letter

### Multiple-Address Letter
- **Chapter:** Ch 8
- **Letterhead:** YES
- **Numbered Paragraphs:** YES (8 levels)
- **Use:** Same letter to multiple action addressees

### Executive Memorandums (HqDON/OSD)

| Type | Use |
|------|-----|
| **Standard Memo** | General HQ correspondence |
| **Action Memo** | Request decision/approval |
| **Information Memo** | Brief senior leadership |
| **Decision Memo** | Present options for decision |

## Quick Comparison

| Document | Numbered Para | From/To | Salutation | Signature Style |
|----------|---------------|---------|------------|-----------------|
| Naval Letter | YES | YES | NO | Abbreviated name |
| Business Letter | **NO** | NO | YES | Full name, rank, title |
| Endorsement | NO* | YES | NO | Abbreviated name |
| MOA/MOU | YES | NO | NO | Overscored full name |
| Joint Letter | YES | YES | NO | Dual signatures |

*Endorsements continue the basic letter's paragraph sequence

## Page Numbering

| Page | Format |
|------|--------|
| First page | NOT numbered |
| Second+ | Centered, 0.5" from bottom, starting with "2" |

---

## Signature Block Position

- **Naval Letter:** Center of page, 4th line below text
- **Business Letter:** Center, 4th line below "Sincerely,"
- **MOA/MOU:** Dual columns, overscored

---

## Classification Markings

If classified, include:
- Classification level in header/footer
- Classified by / Derived from / Reason / Declassify on block

If CUI:
- Controlled by / CUI Category / Dissemination / POC block
- Distribution statement
